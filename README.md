## CultureByte - A Food Platform

### Overview
Culturebyte is a online food ecommerce platform which provides statewise wide variety of food options.

### Key Features
* Statewise and Culturewise Food Options
* Responsive User Interface
* Stripe Payment Gateway Integration
* Redux State Management
* Clerk based User Authentication

### Technologies Used
* Frontend: React
* Backend: Express
* Database: MongoDB
* State Management: Redux
* Payment Gateway: Stripe API
* User Auth: Clerk
